# Wordparss 範本檔案上傳檔案管理外掛
主要功能可以在後台上傳範本檔案，範本檔案可以在頁面編輯時在範本中選擇使用。

## 目前主要功能
-- 後臺上傳頁面與範本檔案列表。
-- 可以刪除上傳的範本檔案。
-- 在頁面編輯畫面可以選擇上傳範本。
